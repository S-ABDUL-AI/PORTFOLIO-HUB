# Civic Tech Portfolio

[![Open Portfolio](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://civic-tech-portfolio.streamlit.app)

Production-ready decision-support tools for SNAP program officers, Medicaid administrators, global health funders, and public-sector analytics teams. Each app opens directly in the browser — no Streamlit account required.

---

## Featured Tools

| Tool | Description | Open App |
|------|-------------|----------|
| **Medicaid & Healthcare Access Risk Monitor** | State-level coverage risk scoring for all 50 US states — insurance gaps, cost burden, income capacity, and rural reach. Structured policy briefs for Medicaid program officers and federal policy teams. | [Open App](https://medicaid-healthcare-access-risk-monitor.streamlit.app) |
| **Safety Net Risk Monitor** | Proactive SNAP and food security vulnerability targeting — identifies communities at highest risk before crisis point, with policy briefs and recommended actions. | [Open App](https://safety-net-risk-monitor.streamlit.app) |
| **Public Budget Allocation Tool** | Need-based government budget distribution across regions. Generates ministerial-grade decision briefs with risk flags and immediate action steps. | [Open App](https://smart-resource-allocation-dashboard.streamlit.app) |
| **GovFund Allocation Engine** | Cost-effectiveness decision tool for global health funders. Models cost-per-life-saved across malaria, nutrition, and social protection interventions. | [Open App](https://govfund-allocation-engine.streamlit.app) |

## Run Locally

```bash
git clone https://github.com/S-ABDUL-AI/CIVIC-TECH-PORTFOLIO.git
cd CIVIC-TECH-PORTFOLIO
pip install streamlit
streamlit run app.py
```
